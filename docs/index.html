hAI.OpenCodeHAI

OpenCode AI Coding Agent als Docker/Portainer-Stack –
nutzbar im Browser, auf der Konsole und als Remote-API für 9Router.

📦 GitHub Repo 📖 Dokumentation 🔗 MCP-Setup 🌐 opencode.ai
🎯 3 Nutzungsziele
🌐

Browser-UI

OpenCode direkt im Browser – kein Terminal nötig.

http://<server-ip>:4096
💻

Konsole direkt

Einfach opencode tippen – Alias erledigt den Rest.

alias opencode="docker exec..."
🔌

Remote-API für 9Router

OpenCode als Provider in 9Router einbinden.

http://<server-ip>:4096/v1
⚙️ .env Konfiguration

📄 .env Datei (niemals committen!)

# AI Provider
OPENAI_API_KEY=sk-...
OPENAI_BASE_URL=https://9router.example.com/v1
OPENCODE_MODEL=deepseek/deepseek-chat

# GitHub MCP (optional)
# Token: https://github.com/settings/tokens
# Scopes: repo, read:org
GITHUB_TOKEN=github_pat_...
⚠️ Die .env Datei niemals in Git committen – sie steht bereits in .gitignore.

🚀 Schnellstart
1

Repo klonen

git clone https://github.com/jbkunama1/hAI.OpenCodeHAI.git
cd hAI.OpenCodeHAI && cp .env.example .env
2

.env befüllen

OPENAI_BASE_URL=https://9router.example.com/v1
OPENAI_API_KEY=sk-...
OPENCODE_MODEL=deepseek/deepseek-chat
GITHUB_TOKEN=github_pat_...  # optional für MCP
3

Stack starten

docker compose up -d && docker logs -f opencode
4

Browser öffnen

http://<server-ip>:4096
5

Konsolen-Alias setzen

echo 'alias opencode="docker exec -it opencode opencode"' >> ~/.bashrc
source ~/.bashrc
