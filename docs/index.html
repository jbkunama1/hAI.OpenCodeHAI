hAI.OpenCodeHAI

OpenCode AI Coding Agent als Docker/Portainer-Stack auf DietPi & Debian – kostenlos, quelloffen, modellflexibel.

📦 GitHub Repo 📖 Dokumentation 🌐 opencode.ai
✨ Features
🐳

Docker-Ready

Sauberer Container, persistente Volumes, Portainer Stack – sofort einsatzbereit.

🆓

Kostenlos nutzbar

DeepSeek, Gemini Flash und Qwen bieten großzügige Free-Tiers – kein Abo nötig.

🔀

Multi-Modell

Claude, GPT, Gemini, DeepSeek, Qwen, Ollama – frei wählbar per ENV-Variable.

🍓

DietPi optimiert

Schlanker Footprint, perfekt für ARM-Server und ressourcenarme Umgebungen.

🔒

Sicher

API-Keys nur in .env, nie im Repository. .gitignore schützt sensible Daten.

🔄

Auto-Update

Watchtower-Label für automatische Container-Updates vorkonfiguriert.

🚀 Schnellstart
1

Repo klonen

git clone https://github.com/jbkunama1/hAI.OpenCodeHAI.git
cd hAI.OpenCodeHAI
2

.env anlegen

cp .env.example .env
# DEEPSEEK_API_KEY=sk-...  eintragen
# OPENCODE_MODEL=deepseek/deepseek-chat
3

Stack starten

docker compose up -d
4

OpenCode nutzen

docker exec -it opencode sh
cd /workspace && opencode
